# AI-Assistance Disclosure Draft

This is a draft to review with the supervisor before using it in the thesis.

## Greek draft

Στο πλαίσιο της παρούσας διπλωματικής εργασίας χρησιμοποιήθηκαν εργαλεία τεχνητής νοημοσύνης ως βοηθητικά μέσα για την οργάνωση του έργου, τη συγγραφή αρχικών προσχεδίων, την αναζήτηση πιθανών σφαλμάτων στον κώδικα και τη βελτίωση της σαφήνειας του κειμένου. Η τελική ευθύνη για το περιεχόμενο, την ορθότητα των υλοποιήσεων, την εγκυρότητα των πειραματικών αποτελεσμάτων, την ακρίβεια των βιβλιογραφικών αναφορών και τη διατύπωση των συμπερασμάτων ανήκει αποκλειστικά στον συγγραφέα της εργασίας. Όλες οι τεχνικές αξιώσεις, τα αποτελέσματα και οι βιβλιογραφικές αναφορές πρέπει να έχουν ελεγχθεί ανεξάρτητα πριν από την τελική υποβολή.

## English draft

Artificial intelligence tools were used as auxiliary tools during the preparation of this thesis, mainly for project organization, initial drafting, code review assistance, and improving textual clarity. The final responsibility for the content, correctness of the implementations, validity of the experimental results, accuracy of the bibliography, and conclusions remains solely with the author. All technical claims, results, and references must be independently verified before final submission.
